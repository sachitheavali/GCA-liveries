### Boeing 737 MAX 8 (B38M) — iFly Livery Pack

**Version:** 1.2

> **Compatibility Notice:** This livery pack is exclusively for the **iFly 737 MAX 8 (178 Seats)** and **PMDG 737-800** on **MSFS 2020**. It will not work with other variants.

---

## Contents

| Folder | Description |
|---|---|
| `B38M 737-MAX8` | Folder containing liveries for the iFly 737-MAX8 |
| `B738 737-800` | Folder containing liveries for the PMDG 737-800 |

| Folder | Description|

## Installation

### 1. Liveries
- Take your livery(s) of choice and paste them into your **Community** folder.

### 2. Configs
- For the ifly 737-MAX8
  - Copy the **contents** of the `Configs` folder — *do not copy the folder itself*.

  - Paste into your **work** folder:
    ```
    C:\Users\{Youruserprofile}\AppData\Local\Packages\Microsoft.FlightSimulator_8wekyb3d8bbwe\LocalState\packages\ifly-aircraft-737max8\work
    ```
- For the PMDG 737-800
  - Copy the **contents** of the `Configs` folder — *do not copy the folder itself*.

  - Paste into your **work** folder:
    ```
    C:\Users\{Youruserprofile}\AppData\Local\Packages\Microsoft.FlightSimulator_8wekyb3d8bbwe\LocalState\packages\pmdg-aircraft-738\work\Aircraft
    ```
### 3. Pictures
- Save the images and insert them into your tracking software.

### 4. SimBrief Profiles
1. Copy the provided link from the text file and paste it into your browser.
2. Ensure you're signed into your SimBrief profile.
3. Click **"Save Airframe."** at the top left of the page.
